Objectives

To provide an opportunity for students to apply the analytical knowledge, practical skills and software tools learned in this course to a selected real-world problem to help decision-makers and stakeholders make clear and defensible decisions.

The Project Proposal (Due Friday Week 9)

The one-page proposal should have the following information:

  • Your name, matriculation number and contact.
  • Title of term paper (tentative, but be specific as possible).
  • Problem background, description and objectives.
  • Decisions to be made.
  • Possible alternatives for each decision above.
  • Major factors and uncertainties.
  • Sources of data/information.
  • Software: DPL, YAAHP, Excel, Py, etc.
  • Other information.

Final Report

  • Cover Page
  • Abstract or Summary
  • Table of Contents
  1. Problem Description
  2. Base Model Formulation
  3. Deterministic Structuring
  4. Probabilistic Evaluation
  5. Risk Aversion Analysis
  6. What-If, Scenario & Options Analysis
  7. Consideration of Other Factors (If any)
  8. Recommendations & Conclusions
  • References (if any)
  • Appendix A (if any)
  • Appendix B (if any)
  • ...

Project Deliverables (Due Monday Reading Week)

  • Project Report (PDF or Word)
  • Model files (DPL, YAAHP, Excel etc) and computer code (.py, .jpynb) if any
  • 1-slide PPT poster (Template)
       Do not change the aspect ratio or orientation of the poster
       Submit the PPT file, do not convert to PDF
  • Pack all the above items into ONE ZIP file and name it <Name1-Name2>.zip
