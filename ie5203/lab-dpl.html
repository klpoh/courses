Installation of DPL 9 Software

You may install DPL 9 for this lab and for doing your term paper the following ways:

  1. Request to install an Academic Version of DPL9 (Professional) by signing and submitting this Installation Agreement Form via Canvas Assignment. Installation instructions will be sent to you by the TA.
  2. Alternatively, but not recommended, you may request for a trial version of DPL9 (Professional) directly from Syncopation Software. Note that the trial version is fully functional but is valid only for a very limited time period which may not cover your term paper time window.

 

Note that this is a Windows application software. Mac users may find this LINK useful to learn how to run Windows software on Mac.
