Project Groups

Objectives

The group project provides opportunities for students to integrate and apply the analytical knowledge and computational skills learned in this course. Students will work in small groups to model and analyze a selected relevant real-world problem using computer software to help decision-makers and stakeholders make informed economic or financial decisions.

Project Schedule, Milestones and Deliverables

  • Formation of Project Groups (Week 3, completed)
    Submit your proposed project group using this Word Template via Canvas by 2 Feb 2024. (Closed)
  • Submission of a 1-page proposal (Week 5, completed)
    Submit your group's project proposal using this Word Template via Canvas by 16 Feb 2024. (Closed)
  • Submission of Progress Report (Extended to Week 8 Tuesday, completed)
    Submit your group's project progress report via Canvas by 12 March 2024. (closed)
  • Submission of final reports and deliverables (Week 13 Monday)
    Submit your group's final report and deliverables in one ZIP file via Canvas by 15 April 2024, 5pm.
    See details below

One-Page Project Proposal

A one-page proposal using this Submission Template is to be submitted via Canvas by 16 Feb 2024. It should contain but not limited to the following information:

  • Group members names and contacts
  • Tentitative project title or topic
  • Problem description and objectives
  • Decisions to be made
  • Possible alternatives for each decision above
  • Major factors and uncertainties
  • Possible sources of data/information
  • Any other information

Project Progress Report (Week 7/8)

Students are expected to have made good progress on the project at least up to the deterministic base-value cash flow modeling phase. The project progress report should contain but not be limited to the following:

  • Problem description including key assumptions
  • Decisions and their alternatives
  • Data for base-value scenarios
  • Cash flow model for each alternative based on base-value data
  • Best or optimal alternative for base-value scenario
  • Proposed further work and schedule (Gantt Chart)

 

Progress report including Excel file (if any) should be zipped into one single file, and uploaded to Canvas.

Only one submission is required per group

Name the uploaded file: <Group_No>-progress-report.ZIP

Submission of Final Report and Deliverables (Week 13)

The following are due for submission by 15 April 2024 5pm via Canvas .

  • Final Project Report (PDF or Word)
  • Excel files, PY, Jpynb, R codes, etc if any
  • A one-slide poster (PPT Poster Template)
    Submit original PPT file and NOT PDF version
    Do not change the aspect ratio or orientation of the poster

 

All deliverables should be ZIPPED into ONE single file and named <Group_No>-project-report.ZIP

The Final Report

The final report should be written in a formal report style and citations (if any) should be in APA style. The report is expected to contain but not limited to:

  • Cover Page with group members' name and contact.
  • Abstract/Summary
  • Table of Contents
  1. Problem description including key assumptions
  2. Decision alternatives
  3. Base-value Data for each alternative
  4. Finanical or cash flow model for each alternative
  5. Best alternative and solutions based on base-value data
  6. Understanding key uncertainties
  7. Probablistic risk analysis
  8. What-If & Scenario Analysis
  9. Further Analysis: Consideration of non-financial factors if any
  10. Recomendations and Conclusion
  • Refereces
  • Appendix A
  • Appendix B
  • ...

Group Presentations

Presentations are schedule for 15, 16 and 17 April 2024. Book your group's presentation slot on Canvas Calendar.

Each group will do a 10-minute presentation followed by a 5-minute Q&A.

Submission of Presentation Slides

Upload your presentation slides (pptx) to Canvas Folder within 24 hours after your presentation. You are strongly encouraged to attend all the sessions to learn from the other groups.
